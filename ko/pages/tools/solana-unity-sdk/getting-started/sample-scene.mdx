## 기능

- Prefabs는 객체 설정을 저장하는 템플릿입니다. 같은 자산의 인스턴스를 만들 수 있고 재사용도 쉽습니다
- Scenes 폴더에는 여기 배포된 전체 데모가 포함되어 있습니다
- Scrips 폴더에는 다양한 스크립트 예제가 포함되어 있습니다(예: Orca 스왑용 `Assets/Samples/Solana SDK/0.1.1/Sample Wallet/Solana Wallet/Scripts/example/screens/SwapScreen.cs`)
- Animations, Materials, Plugins, Textures는 이름 그대로의 용도를 가집니다

이 데모 씬을 직접 다뤄 보면서 지갑, NFT 로딩, 전송, 스왑 등의 기본 기능에 익숙해져 보세요.
