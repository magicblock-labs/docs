## Unity

[여기](https://unity.com/download)에서 Unity를 설치하세요.

## SDK 패키지 가져오기

- [Unity Package Manager](https://docs.unity3d.com/Manual/upm-ui.html) 창을 엽니다.
- 상태 표시줄의 `+` 버튼을 클릭합니다.
- 패키지 추가 옵션이 나타납니다.
- 추가 메뉴에서 `Add package from git URL`을 선택합니다. 텍스트 상자와 `Add` 버튼이 표시됩니다.
- ![](/images/package_manager.png)
- 텍스트 상자에 `https://github.com/magicblock-labs/Solana.Unity-SDK.git` Git URL을 입력한 뒤 `Add`를 클릭합니다.
- 패키지가 설치되면 Package Manager 검사기에서 `Samples`를 볼 수 있습니다. `Import`를 클릭하세요.
- 특정 버전을 지정한 URL을 사용해 원하는 패키지 버전을 설치할 수도 있습니다.

  - `https://github.com/magicblock-labs/Solana.Unity-SDK.git#X.Y.X`
  - 여기 적힌 `X.Y.Z`는 설치하려는 버전 번호로 바꿔야 합니다.
  - 사용 가능한 모든 릴리스는 [여기](https://github.com/magicblock-labs/Solana.Unity-SDK/releases)에서 확인할 수 있습니다.
  - 최신 릴리스 버전은 [![Last Release](https://img.shields.io/github/v/release/magicblock-labs/Solana.Unity-SDK)](https://github.com/magicblock-labs/Solana.Unity-SDK/releases/latest) 입니다

- 샘플 씬 가져오기
- ![](/images/sample_scene.png)
- 지갑이 설정된 샘플 씬은 `Samples/Solana SDK/0.0.2/Simple Wallet/Solana Wallet/scenes/wallet_scene.unity` 에서 찾을 수 있습니다.
