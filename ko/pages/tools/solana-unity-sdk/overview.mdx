## Solana.Unity-SDK

Solana.Unity SDK는 Unity 기반 게임에서 Solana에 쉽게 접근할 수 있도록 해 주는 포괄적인 오픈소스 도구 모음입니다. SDK는 Unity Package Manager 또는 Unity Asset Store를 통해 설치할 수 있으며, 제공되는 옵션 중에서 원하는 지갑을 설정할 수 있습니다. Solana.Unity-SDK는 [Solana.Unity-Core](https://github.com/garbles-labs/Solana.Unity-Core) 구현을 기반으로 하며, 네이티브 .NET Standard 2.0(Unity 호환), 완전한 RPC API 커버리지, MPL, 네이티브 DEX 작업 등을 제공합니다.

## 기능

- 완전한 JSON RPC API 커버리지
- 지갑 및 계정: Unity 안에서 비수탁형 Solana 지갑 설정(sollet 및 solana-keygen 호환)
- Phantom 및 Web3auth 지원(소셜 계정을 통한 비수탁형 가입/로그인)
- base64 및 wire format에서 트랜잭션 디코딩 후 wire format으로 재인코딩
- base64 및 wire format에서 메시지 디코딩 후 wire format으로 재인코딩
- 명령어 디컴파일
- SPL token 송수신과 Associated Token Accounts의 JIT 프로비저닝을 위한 TokenWallet 객체
- 기본 UI 예제
- NFTs
- 게임을 xNFT(Backpack)로 빌드
- 네이티브 DEX 작업(Orca, Jupiter는 곧 지원 예정...)
- WebSocket으로 사용자 정의 이벤트 등록/트리거(계정 변경, 서명 상태, 프로그램 등)
- Solana Mobile Stack 지원
- Solana Wallet Adapter
