## Magic Router SDK

Magic Router SDK는 MagicBlock의 Ephemeral Rollups 인프라에서 트랜잭션 준비와 라우팅을 단순화하는 TypeScript 라이브러리입니다. 트랜잭션을 Solana 메인넷에서 실행할지 Ephemeral Rollup에서 실행할지 수동으로 판단하는 대신, SDK가 메타데이터를 분석해 이 복잡성을 자동으로 처리합니다.

## 목적과 장점

Magic Router SDK는 MagicBlock의 이중 레이어 실행 환경이 가진 복잡성을 추상화합니다.

- **자동 라우팅**: 계정 메타데이터를 기반으로 트랜잭션을 Ephemeral Rollups 또는 Solana로 지능적으로 라우팅합니다
- **단순한 통합**: 단일 엔드포인트 연결만으로 충분하며 수동 라우팅 로직이 필요 없습니다
- **최적화된 성능**: 가장 빠른 사용 가능 레이어에서 트랜잭션을 실행해 즉시 확인을 노립니다
- **매끄러운 지갑 경험**: 표준 Solana 트랜잭션 서명을 유지하면서 향상된 라우팅 기능을 제공합니다
