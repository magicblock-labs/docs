## 수수료

MagicBlock의 요금 체계는 탈중앙화 클라우드 인프라와 유사합니다. 이 프로토콜은 애플리케이션 개발자가 컴퓨팅 비용을 예측 가능하게 유지할 수 있도록 설계되었으며, 동시에 원하면 직접 노드를 운영하고 우선 접근에 대한 수수료를 설정할 수 있는 유연성도 제공합니다. 요금 체계는 Public nodes와 Dedicated nodes 두 가지입니다.

### Public Nodes

Public nodes는 누구나 마찰 없이 MagicBlock에서 무료 트랜잭션으로 개발을 시작할 수 있게 해줍니다. Session fee는 ER 세션마다 부과되며 undelegation 시 적용됩니다. Commit fee는 계정을 Solana로 커밋할 때마다 부과됩니다.

| **수수료 유형** | **금액 (SOL)** | **설명** |
| :------------- | :------------- | :------- |
| **기본 수수료** | 0              | 거래당 |
| **세션 수수료** | 0.0003         | ER 세션당 |
| **커밋 수수료** | 0.0001         | 커밋당 |

### Dedicated Nodes

Dedicated nodes는 기업과 대규모 팀에 최고 수준의 안정성과 서비스 품질을 제공합니다. 전용 노드로 MEV를 방지하고 예측 가능한 비용의 이점을 누릴 수 있습니다。

| **수수료 유형** | **금액 (SOL)** | **설명** |
| :------------- | :------------- | :------- |
| **기본 수수료** | 0.0000005      | 거래당 |
| **세션 수수료** | 0.0003         | ER 세션당 |
| **커밋 수수료** | 0.0001         | 커밋당 |

사용량 기반 할인이나 자체 노드 운영에 관심이 있다면 [development@magicblock.xyz](mailto:development@magicblock.xyz) 로 문의해 주세요.
