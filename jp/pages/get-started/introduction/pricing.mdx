## 料金

MagicBlock の料金体系は分散型クラウドインフラに近い考え方です。アプリケーション開発者にとって計算コストを予測しやすく保ちながら、必要に応じて自前のノードを運用したり、優先アクセスに対して料金を設定したりできる柔軟性も提供します。料金体系は Public nodes と Dedicated nodes の 2 種類があります。

### Public Nodes

Public nodes では、誰でも摩擦なく MagicBlock 上で無料トランザクションを使って開発を始められます。Session fee は ER セッションごとに発生し、undelegation 時に適用されます。Commit fee は、アカウントを Solana にコミットするたびに発生します。

| **料金種別** | **金額 (SOL)** | **説明** |
| :----------- | :------------- | :------- |
| **基本料金** | 0              | 取引ごと |
| **セッション料金** | 0.0003    | ER セッションごと |
| **コミット料金** | 0.0001      | コミットごと |

### Dedicated Nodes

Dedicated nodes は、企業や大規模チーム向けに最大限の信頼性とサービス品質を提供します。専用ノードにより MEV を防ぎつつ、予測しやすいコストの恩恵を受けられます。

| **料金種別** | **金額 (SOL)** | **説明** |
| :----------- | :------------- | :------- |
| **基本料金** | 0.0000005      | 取引ごと |
| **セッション料金** | 0.0003    | ER セッションごと |
| **コミット料金** | 0.0001      | コミットごと |

ボリュームディスカウントや独自ノードの運用については、[development@magicblock.xyz](mailto:development@magicblock.xyz) までご連絡ください。
