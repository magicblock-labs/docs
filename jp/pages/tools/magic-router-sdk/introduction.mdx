## Magic Router SDK

Magic Router SDK は、MagicBlock の Ephemeral Rollups 基盤におけるトランザクション準備とルーティングを簡素化する TypeScript ライブラリです。トランザクションを Solana メインネットで実行するのか、Ephemeral Rollup で実行するのかを手動で判断する代わりに、SDK がメタデータを解析して自動的に処理します。

## 目的と利点

Magic Router SDK は、MagicBlock の二層実行環境にある複雑さを抽象化します。

- **自動ルーティング**: アカウントメタデータに基づいて、取引を Ephemeral Rollups または Solana に賢く振り分けます
- **統合の簡素化**: 単一エンドポイントへの接続だけで済み、手動のルーティングロジックは不要です
- **最適化された性能**: 取引は最も高速な利用可能レイヤーで実行され、即時確認を目指します
- **シームレスなウォレット体験**: 標準的な Solana トランザクション署名を維持しつつ、強化されたルーティング機能を提供します
