## Magic Router SDK

Magic Router SDK 是一个 TypeScript 库，用于简化 MagicBlock Ephemeral Rollups 基础设施中的交易准备与路由。你无需手动决定交易应在 Solana 主网还是 Ephemeral Rollup 上执行，SDK 会通过智能元数据分析自动处理这部分复杂性。

## 目的与优势

Magic Router SDK 抽象掉了 MagicBlock 双层执行环境中的复杂性：

- **自动路由**：根据账户元数据，智能将交易路由到 Ephemeral Rollups 或 Solana
- **简化集成**：只需连接单个端点，无需手写路由逻辑
- **性能优化**：交易会在当前最快的可用层上执行，以获得即时确认
- **无缝钱包体验**：保持标准 Solana 交易签名方式，同时增强路由能力
