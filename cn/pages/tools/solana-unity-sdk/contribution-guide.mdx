## 寻找可参与的问题

如果你想开始参与，可以先查看[开放中的 issues](https://github.com/magicblock-labs/Solana.Unity-SDK/issues)。对于简单的文档修改或拼写错误，欢迎直接提交 pull request。

如果你计划进行较大的改动或主动推动某项新功能，请先提交一个 issue，并在 [Telegram](https://t.me/+78KHQkUsy0ViMzQ6) 中与维护者沟通。

## 选择一个 issue

如果你想参与贡献，请通过评论、fork 仓库并创建一个 pull request（即使是空的也可以）来认领 issue。这样维护者就能跟踪谁正在处理哪个问题，避免重复劳动。

## 参与规范

请遵循以下准则：

开始编码前：

- 选择一个能描述你正在处理问题的分支名

编码过程中：

- 尽快提交一个 draft PR
- 只修改与你的 PR 直接相关的代码。有时你可能会发现某些代码确实需要重构，但如果它与当前 PR 无关，请不要顺手修改，改为提交一个 issue。这样能让审阅者一次专注于一个问题。
