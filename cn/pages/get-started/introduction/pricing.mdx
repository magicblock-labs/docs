## 费用

MagicBlock 的定价方式类似于去中心化云基础设施。该协议旨在让应用开发者的计算成本保持可预测，同时也提供自行运行节点的灵活性，并可在需要时为优先访问设置费用。目前提供两类定价：Public nodes 和 Dedicated nodes。

### Public Nodes

Public nodes 让所有人都能以免费交易、零门槛的方式开始在 MagicBlock 上构建。Session fee 按每次 ER 会话收取，并在 undelegation 时执行。Commit fee 则会在你每次将账户提交回 Solana 时收取。

| **费用类型** | **金额 (SOL)** | **说明** |
| :----------- | :------------- | :------- |
| **基础费用** | 0              | 每笔交易 |
| **会话费用** | 0.0003         | 每个 ER 会话 |
| **提交费用** | 0.0001         | 每次提交 |

### Dedicated Nodes

Dedicated nodes 为企业与大规模团队提供最高级别的可靠性和服务质量。你可以通过专属节点防范 MEV，并获得更可预测的成本。

| **费用类型** | **金额 (SOL)** | **说明** |
| :----------- | :------------- | :------- |
| **基础费用** | 0.0000005      | 每笔交易 |
| **会话费用** | 0.0003         | 每个 ER 会话 |
| **提交费用** | 0.0001         | 每次提交 |

如需基于使用量的折扣，或希望自行运行节点，请联系 [development@magicblock.xyz](mailto:development@magicblock.xyz)
